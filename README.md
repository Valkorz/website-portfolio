**Why Astro?**
Choosing astro â€” alongisde **Tailwind CSS** â€” was a deliberate choice, due to my demand for a highly responsive, attractive and performant website. Astro uses the "Islands Architecture" design pattern, which renders the entire website as a static HTML page on the server, only sending JavaScript when necessary. This turned out as an amazing choice for my portfolio needs. Tailwind CSS was also a great match, due to its ease of use and great reputation.
Combining both Astro and Tailwind CSS allowed me to easily develop a well-decorated user interface whilst avoiding pollution caused by style definition, increasing readability and maintainability.
