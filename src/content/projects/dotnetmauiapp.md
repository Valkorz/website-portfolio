# INTRODUCTION

An application that simulates a student platform for FACENS, allowing students to check their grading status and classes. Features an admin login that allows the creation of new users and classes in the database.

# IMPLEMENTATION

The project was developed using Dotnet MAUI and Entity Framework. MAUI is a modern solution for developing performant multi-platform apps.
